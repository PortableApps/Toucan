PortableApps.com Updater 1.0.0.1 Alpha Help

Portable Apps up to date

PortableApps Update is created with some ideas and code from the PortableApps.com (R) Installer from John Haller.

It uses GNU Wget Version 1.11 as a binary version found on: http://wget.addictivecode.org/
(This version was compressed with upx --best --compress-icons=0 --crp-ms=999999 in order to save space.)

PortableApps.com Updater is a tool designed to gets Your PortableApps up to date and it is published under the terms of GPL. It fits with the PortableApps.com (R) Platform 1.x but works propper without it too.

Keep in mind that this is a stable but ALPHA state version. Everything (!) could change in future versions, depending from the current PortableApps.com &TM; PlatformOnly PortableApps with the PAF (PortableApps.com (R) Format) as Installer (newer Apps) or as self extracting archives are supported.

If You'll use packages outside PortableApps.com You'll do this at Your own risc. There is no help or support for this packages!

PA_UP needs access to the internet. It supports proxy connections (see here)

It checks for PortableApps folders in the same directory as the PortableAppsUpdater folder. For all supported Apps (at the moment all published Apps on portableapps.com) which are not present or different as the published versions download and installation is offered. A messagebox informs if all versions are up to date.

If You don't want a supported App, set it on Your personal blacklist in Your settings.ini and it will never checked untill You'll enable it again.
