Secure

With Secure you can encrpyt and decrypt your files using AES-256, a secure encryption system.

Jobs

You can use the drop-down box and buttons to create new Secure jobs, save your current data as a job and remove existing ones. Existing jobs will appear in th drop-down box. Jobs can also be used on the command line as well as in the user interface. Jobs allow you to save you data and reload it when you next run Toucan.

Function

Allows you to set whether to Encrypt or Decrypt the files in the file list.

Action Buttons

OK runs Secure with the currently entered information. Preview will change the colours of the items that you had added to the file list. If an item remains black then it will be encrypted/decrypted. If the item is changed to red then it will not be encrypted/decypted. Previewing does not effect the files in any way.

Rules

You can use this drop-down box to select any rules that apply when Secure runs. To check they are working as you require it is recommended that you preview them first.

File Browser

The file browser allows you to add files to the file list or to remove them. Double clicking on drives or folders expands them or collapses them. Both files and folders can be added to the file list.

File List

Any files or folders that you add from the browser will appear on this list. When previewing the file names will change, see Action buttons for more information. The upper most button shows the Insert Variable Dialog. The lower button expands all of the folders in the file list so you can quickly see all of the files that you have added.
