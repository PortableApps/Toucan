Backup All Documents

If you have multiple user accounts on one machine then backing up all of their documents is easy with Toucan, you need the following set of Rules:

Folders to exclude:

\Users (or \Documents and Settings if you use XP or below)

Locations to Include

\Documents (or \My Documents if you use XP or below)

Simply save this into a set of rules and then use it with a job that has the Users or Documents and Settings directory in it and only the documents will be backed up.
