Standard Variables

These can be used in any path by wrapping them in @ signs, e.g. @date@.

  • date - The current date in the format yyyy-mm-dd
  • time - The current time in the format hh-mm
  • YYYY - The year in its four digit form
  • MM - The two digit month
  • DD - The current day
  • hh - The current hour
  • mm - The current minute
  • drive - The drive letter that Toucan is being run from
  • docs - The My Documents folder
  • volume - The label of the drive Toucan is being run from
  • label - The label from an autorun.inf file, if it exists

Windows environment variables can also be used by wrapping then in @ signs, e.g. @PATH@
